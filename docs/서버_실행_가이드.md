# 서버 실행 가이드

## 사전 준비

- Python 3.9 이상이 설치되어 있어야 합니다.
- `.env` 파일에 `OPENAI_API_KEY`가 설정되어 있어야 합니다.
- 가상환경(`.venv`)이 생성되어 있어야 합니다.

가상환경이 없다면 최초 1회 아래 명령어를 실행해 주세요:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

---

## 서버 켜기

터미널을 열고 프로젝트 폴더로 이동합니다:

```bash
cd /Users/ryong/Documents/smart_chatbot
```

가상환경을 활성화합니다:

```bash
source .venv/bin/activate
```

> 프롬프트 앞에 `(.venv)`가 표시되면 활성화된 상태입니다.
> 이미 `(.venv)`가 표시되어 있다면 이 단계를 건너뛰세요.

서버를 실행합니다:

```bash
python main.py
```

아래 메시지가 나오면 정상 실행된 것입니다:

```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
INFO:     Application startup complete.
```

---

## 서버 끄기

서버가 실행 중인 터미널에서 `Ctrl + C`를 누르면 종료됩니다.

---

## 접속 방법

| URL | 설명 |
|-----|------|
| http://localhost:8000 | 서버 상태 확인 |
| http://localhost:8000/docs | Swagger UI (API 테스트) |

---

## API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/` | 서버 상태 확인 |
| GET | `/models` | 사용 가능한 GPT 모델 목록 조회 |
| POST | `/chat` | 챗봇에게 메시지 전송 |

### POST /chat 요청 예시

```json
{
  "message": "안녕하세요!",
  "model": "gpt-4o-mini"
}
```

> `model`은 생략 가능하며, 기본값은 `gpt-4o-mini`입니다.
