# 탐색 및 검색

탐색 페이지는 하단 네비게이션의 두 번째 탭에 해당하며, 벙을 찾고 검색할 수 있는 페이지입니다.

## 탐색 홈

### 표시 정보
- **"탐색"** 제목이 상단에 표시됩니다
- **검색 바**: "벙 검색" 플레이스홀더가 있는 검색 버튼
- **추천 벙 리스트**: 하단에 참여 가능한 추천 벙 목록이 표시됩니다

### 사용자 액션
- 검색 바를 탭하면 검색 모드로 전환됩니다
- 추천 벙 카드를 탭하면 해당 벙의 상세 페이지로 이동합니다

---

## 검색 기능

### 검색 방법
1. 검색 바를 탭하면 검색 입력란이 자동 포커스됩니다
2. 최소 **2글자 이상** 입력해야 검색이 실행됩니다
3. 입력 후 잠시 대기하면 자동으로 검색 결과가 표시됩니다
4. **"취소"** 버튼으로 검색을 종료하고 탐색 홈으로 돌아갑니다

### 검색 탭 (4가지 필터)

#### 1. 전체 탭
- 멤버, 해시태그, 위치 3개 카테고리의 검색 결과를 각각 최대 4개씩 미리보기로 표시합니다
- 각 카테고리 제목에 검색 키워드가 파란색으로 하이라이트됩니다
- 카테고리 제목을 탭하면 해당 탭으로 이동합니다

#### 2. 멤버 탭
- 닉네임으로 사용자를 검색합니다
- 검색된 멤버의 프로필 이미지, 닉네임, 좋아요 수가 표시됩니다
- 해당 멤버가 참여 중인 벙 리스트가 함께 표시됩니다

#### 3. 해시태그 탭
- 해시태그로 벙을 검색합니다
- "#키워드 해시태그가 달린 모임" 제목과 함께 해당 해시태그가 포함된 벙 리스트가 표시됩니다

#### 4. 위치 탭
- 위치(장소명, 주소)로 벙을 검색합니다
- "키워드에서 열리는 러닝" 제목과 함께 해당 위치의 벙 리스트가 표시됩니다
- 위치 텍스트에 검색 키워드가 파란색으로 하이라이트됩니다

### 검색 결과 카드
각 검색 결과 카드에는 다음 정보가 표시됩니다:
- 벙 이미지
- 참여 인원 뱃지 (현재/최대)
- 벙 이름
- 장소
- 날짜/시간
- (멤버 탭) 검색된 멤버의 프로필 이미지
- (해시태그 탭) 해시태그 뱃지
- (위치 탭) 위치 텍스트에 키워드 하이라이트

### 상태 처리
- **검색 결과 없음**: "검색 결과가 없어요" 메시지가 표시됩니다
- **무한 스크롤**: 스크롤을 내리면 다음 페이지의 결과가 자동으로 로드됩니다
