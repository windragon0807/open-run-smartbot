# 회원가입 및 로그인

## 로그인

### 로그인 방법
오픈런은 블록체인 스마트 월렛을 통해 로그인합니다. 별도의 아이디/비밀번호가 필요 없습니다.

### 로그인 절차
1. 로그인 화면에서 **"월렛 만들고 시작하기"** 버튼을 탭합니다
2. 스마트 월렛이 자동으로 생성되거나 기존 월렛에 연결됩니다
3. 지갑 주소로 인증이 완료되면 서비스에 로그인됩니다

### 자주 묻는 질문 (로그인 관련)

**Q. 오픈런이 제 개인 정보를 수집하나요?**
- 아니요, 개인 정보를 수집하지 않습니다. 월렛 주소만으로 인증하며, 이름이나 이메일 등을 수집하지 않습니다.

**Q. 로그인 시 금융 거래가 발생하나요?**
- 아니요, 금융 거래가 발생하지 않습니다. 서명은 인증용 디지털 서명일 뿐입니다.

**Q. 왜 서명이 필요한가요?**
- 지갑이 본인의 것임을 확인하는 과정입니다. 비밀번호 대신 더 안전하고 간편한 본인 인증 방식입니다.

## 회원가입

최초 로그인 후 닉네임이 설정되지 않은 경우 자동으로 회원가입 화면으로 이동합니다.

### 회원가입 단계 (총 5단계)

#### Step 0 — 환영 화면
- "당신만의 캐릭터, 함께 달리는 즐거움!"이라는 환영 메시지가 표시됩니다
- **"시작하기"** 버튼을 눌러 가입을 시작합니다

#### Step 1 — 닉네임 설정 (필수)
- 서비스에서 사용할 닉네임을 입력합니다
- 실시간으로 닉네임 유효성을 검사합니다:
  - 이미 사용 중인 닉네임은 사용할 수 없습니다
  - 자음만으로 구성된 닉네임은 사용할 수 없습니다
- 유효한 닉네임을 입력하면 **"다음"** 버튼이 활성화됩니다

#### Step 2 — 평균 페이스 설정 (선택)
- 본인의 평균 러닝 페이스를 설정합니다
- 분(0~20분)과 초(0~59초)를 다이얼 피커로 선택합니다
- 예: 5분 30초 = 1km를 달리는 데 5분 30초 소요
- **"건너뛰기"**로 이 단계를 생략할 수 있습니다

#### Step 3 — 주간 러닝 빈도 설정 (선택)
- 일주일에 몇 번 달리는지 설정합니다
- 0~7회 중 선택합니다
- **"건너뛰기"**로 이 단계를 생략할 수 있습니다

#### Step 4 — 가입 완료
- "{닉네임}님, 지금부터 러닝의 새로운 재미를 발견해요!"라는 완료 메시지가 표시됩니다
- **"홈으로"** 버튼을 누르면 회원가입이 완료되고 홈 화면으로 이동합니다

### 회원가입 시 입력 정보 요약

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 닉네임 | 필수 | 서비스에서 사용할 이름 (중복 불가) |
| 평균 페이스 | 선택 | 1km당 소요 시간 (분'초" 형식) |
| 주간 러닝 빈도 | 선택 | 일주일에 달리는 횟수 (0~7회) |
