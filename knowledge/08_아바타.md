# 아바타 시스템

아바타는 오픈런에서 자신만의 캐릭터를 꾸미는 시스템입니다. 도전과제를 달성하여 획득한 NFT 아이템으로 캐릭터를 커스터마이징할 수 있습니다.

## 아바타 페이지 접근 방법

- 홈 화면 헤더의 **옷 아이콘** 버튼
- 프로필 페이지의 **옷 아이콘** 버튼
- NFT 보상 획득 후 **"아바타 꾸미러 가기"** 버튼

## 아바타 변경 페이지

### 화면 구성

#### 상단 - 아바타 미리보기
- 검정 배경 위에 현재 착용 중인 아바타가 실시간으로 표시됩니다
- 오픈런 로고 워터마크가 함께 표시됩니다
- 아이템을 선택/해제할 때마다 미리보기가 즉시 업데이트됩니다

#### 하단 - 아이템 선택 영역

**카테고리 선택**
아바타 아이템은 다음 카테고리로 분류됩니다:

| 메인 카테고리 | 설명 |
|---------------|------|
| 상의 | 상체 의류 |
| 하의 | 하체 의류 |
| 풀세트 | 전신 세트 의류 |
| 신발 | 러닝화 등 신발류 |
| 얼굴 | 얼굴 표정/스타일 |
| 피부 | 피부색 |
| 머리카락 | 헤어스타일 |
| 액세서리 | 추가 장식 아이템 |

**액세서리 서브 카테고리**
액세서리는 4가지 세부 분류가 있습니다:
- 머리 액세서리 (모자, 헤어밴드 등)
- 눈 액세서리 (선글라스 등)
- 귀 액세서리 (이어폰 등)
- 몸 액세서리

**아이템 리스트**
- 선택된 카테고리에 해당하는 보유 아이템이 3열 그리드로 표시됩니다
- 각 아이템에는 이미지, "NEW" 뱃지(새 아이템), 레어리티 아이콘, 아이템 이름이 표시됩니다
- 선택된 아이템은 흰색 배경과 그림자로 강조됩니다
- 아이템이 없는 경우 "아이템이 없어요. 도전과제를 달성하고 NFT를 획득하세요!" 안내가 표시됩니다

## 아이템 착용/해제

### 착용 방법
- 아이템을 탭하면 해당 아이템이 아바타에 착용됩니다
- 같은 카테고리의 다른 아이템을 탭하면 교체됩니다

### 해제 방법
- 이미 착용 중인 아이템을 다시 탭하면 해제됩니다
- **"초기화"** 버튼을 누르면 모든 착용 아이템이 해제됩니다

### 착용 규칙
- 각 카테고리 슬롯에 1개의 아이템만 착용할 수 있습니다
- 풀세트(전신 세트)와 개별 상/하의는 별도 슬롯입니다
- 액세서리는 4개의 서브 슬롯에 각각 1개씩 착용 가능합니다

## 아이템 희귀도

NFT 아이템에는 3단계 희귀도가 있습니다:

| 등급 | 이름 | 설명 |
|------|------|------|
| Common | 일반 | 가장 흔한 등급 |
| Rare | 레어 | 희귀한 등급 |
| Epic | 에픽 | 가장 희귀한 등급 |

각 아이템에는 희귀도 뱃지/아이콘이 표시됩니다.

---

## 아이템 상세 보기

각 아이템의 info(i) 아이콘을 탭하면 상세 모달이 열립니다:
- 시리얼 넘버
- NFT 이미지
- 레어리티 뱃지
- 아이템 이름
- 카테고리
- 스캐너 페이지로 이동 버튼 (NFT 상세 확인)

## 아바타 저장 및 캡처

### 저장
- **"저장"** 버튼을 누르면 현재 아바타 상태가 저장됩니다
- 저장된 아바타는 홈 화면 헤더, 프로필 등에 반영됩니다

### 캡처
- 저장 시 아바타 이미지가 캡처되어 캡처 모달이 표시됩니다
- 캡처된 아바타 이미지를 원형 프레임 안에서 확인할 수 있습니다
- **"이미지 다운로드"** 버튼으로 PNG 파일로 저장할 수 있습니다
