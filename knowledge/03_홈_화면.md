# 홈 화면

홈 화면은 로그인 후 가장 먼저 보이는 메인 화면입니다. 하단 네비게이션의 첫 번째 탭에 해당합니다.

## 화면 구성

### 1. 헤더 영역

홈 화면 상단에는 사용자 정보와 날씨가 표시됩니다.

#### 표시 정보
- **아바타**: 현재 착용 중인 아바타 캐릭터가 날씨 배경 위에 표시됩니다
- **닉네임**: 사용자의 닉네임이 표시됩니다
- **지갑 주소**: 축약된 블록체인 지갑 주소가 표시되며, 탭하면 클립보드에 복사됩니다
- **현재 위치**: GPS 기반으로 현재 위치(시/구 단위)가 표시됩니다
- **현재 날씨**: 현재 위치의 날씨 아이콘과 기온(°C)이 표시됩니다
- **좋아요 수**: 다른 사용자로부터 받은 좋아요 수가 표시됩니다

#### 날씨 배경
- 날씨 상태(맑음, 흐림, 비, 눈)와 시간대(오전, 오후, 밤)에 따라 배경 그라데이션이 동적으로 변경됩니다
  - 오전(6~16시), 오후(16~20시), 밤(20시~6시)

#### 사용자 액션
- **아바타 변경 버튼**: 옷 아이콘을 탭하면 아바타 변경 페이지로 이동합니다
- **알림 버튼**: 종 아이콘으로 알림을 확인할 수 있습니다
- **지갑 주소 복사**: 주소 영역을 탭하면 클립보드에 복사됩니다

#### 축소 헤더
- 아래로 스크롤하면 헤더가 축소되어 날씨 아이콘, 기온, 위치가 한 줄로 간결하게 표시됩니다

---

### 2. 나의 벙 섹션

현재 진행 중인 내 벙(러닝 모임) 목록이 표시됩니다.

#### 표시 정보
- 현재 진행 중(ONGOING) 상태인 벙 카드가 최대 10개까지 표시됩니다
- 일정이 없을 경우 "아직 일정이 없어요" 메시지가 표시됩니다

#### 벙 카드 정보
각 벙 카드에는 다음 정보가 표시됩니다:
- **대표 이미지**: 벙의 메인 이미지가 카드 배경으로 표시
- **남은 시간 타이머**: 시작까지 남은 시간이 "D일 HH:MM:SS" 형식으로 실시간 카운트다운
- **진행 중 표시**: 시작 시간이 지나면 "Run Started!" 텍스트가 초록색으로 깜빡이며 표시
- **벙 이름**: 벙의 제목
- **장소**: 러닝 장소
- **날짜/시간**: "M월 d일 (요일) 오전/오후 h:mm" 형식
- **거리 및 페이스**: "Xkm 페이스" 형식
- **벙주 표시**: 내가 벙주인 경우 왕관 아이콘이 표시됩니다

#### 사용자 액션
- 벙 카드를 탭하면 해당 벙의 상세 페이지로 이동합니다
- 하단의 **"벙 만들기"** 버튼으로 새 벙을 생성할 수 있습니다

---

### 3. 추천 벙 섹션

참여 가능한 벙 목록이 추천으로 표시됩니다.

#### 표시 정보
- 참여 가능한 벙이 최대 10개까지 카드 형태로 표시됩니다
- 각 카드에는 벙 이미지, 이름, 장소, 날짜/시간, 참여자 아바타(최대 5명), 남은 자리 수, 해시태그가 표시됩니다
- 열린 벙이 없을 경우 "현재 열린 벙이 없어요. 새로운 벙을 만들어 멤버를 모집해 보세요!" 메시지가 표시됩니다

#### 사용자 액션
- 추천 벙 카드를 탭하면 해당 벙의 상세 페이지로 이동합니다

---

### 4. 챌린지 스와이퍼

진행 중인 도전과제가 자동 슬라이드 형태로 표시됩니다.

#### 표시 정보
- 4초 간격으로 자동 재생되는 카드 슬라이더
- 각 슬라이드에는 챌린지 NFT 이미지, 제목, 설명, 진행률 바(%), 종료 날짜가 표시됩니다

#### 사용자 액션
- 좌우 스와이프로 슬라이드를 탐색할 수 있습니다
- 슬라이드를 탭하면 해당 도전과제로 이동합니다

---

## 하단 네비게이션

모든 메인 화면 하단에 고정된 네비게이션 바가 있습니다.

| 순서 | 이름 | 설명 |
|------|------|------|
| 1 | 홈 | 메인 홈 화면 |
| 2 | 탐색 | 벙 탐색 및 검색 |
| 3 | 생성(+) | 벙 만들기 모달 열기 |
| 4 | 도전과제 | 도전과제 목록 |
| 5 | 프로필 | 내 프로필 |

- 가운데 **"+"** 버튼은 페이지 이동이 아니라 벙 만들기 모달을 엽니다
- 현재 선택된 탭은 강조 표시됩니다
