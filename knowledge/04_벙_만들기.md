# 벙(러닝 모임) 만들기

벙은 오픈런의 핵심 기능으로, 러닝 모임을 만들어 다른 사용자들과 함께 달릴 수 있습니다.

## 벙 만들기 접근 방법

- 홈 화면의 **"벙 만들기"** 버튼
- 하단 네비게이션의 가운데 **"+"** 버튼

## 벙 만들기 절차

벙 만들기는 **2단계**로 구성됩니다.

---

### Step 1: 벙 정보 입력

화면 하단에서 올라오는 풀사이즈 시트에서 벙 정보를 입력합니다.

#### 입력 항목

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 대표 이미지 | 자동 | 서비스에서 제공하는 이미지 중 랜덤 배정. "랜덤" 버튼으로 변경 가능 |
| 벙 이름 | **필수** | 벙의 제목 |
| 설명 | 선택 | 벙에 대한 소개 문구 |
| 장소 | **필수** | 주소 검색을 통해 러닝 시작 장소 설정 |
| 상세 주소 | 선택 | 정확한 집합 위치 (예: "2-1번 출구 인근") |
| 시작 날짜 | **필수** | 달력에서 선택 (오늘 이전 날짜 선택 불가) |
| 시작 시간 | **필수** | 시간 선택기에서 선택 (현재 시점 이후) |
| 예상 시간 | **필수** | 러닝 예상 소요 시간 (분 단위) |
| 거리 | **필수** | 목표 러닝 거리 (km 단위) |
| 페이스 | **필수** | 목표 페이스 (분/초 입력, 예: 5'30") |
| 참가 인원 | **필수** | 최대 참가 가능 인원 (2~500명) |
| 뒷풀이 | **필수** | 러닝 후 뒷풀이 여부 ("유"/"무" 선택). "유" 선택 시 뒷풀이 내용 입력 가능 |
| 해시태그 | 선택 | 벙을 설명하는 키워드 태그. 검색으로 추가, 개별 삭제 가능 |

#### 주소 검색 기능
- **"주소 검색"** 버튼을 탭하면 검색 모달이 열립니다
- **현재 위치 버튼**: GPS 위치를 자동으로 주소로 변환합니다
- **검색**: 도로명+건물번호, 지번 주소, 장소명으로 검색할 수 있습니다
- 검색 결과를 선택하면 지도 미리보기가 표시되고, "선택" 버튼으로 확정합니다

#### 유효성 검사
- 필수 항목을 모두 입력해야 벙을 만들 수 있습니다
- 시작 시간이 현재보다 과거이면 에러가 표시됩니다
- 종료 시간은 시작 시간 + 예상 시간으로 자동 계산됩니다

---

### Step 2: 멤버 초대

벙 생성이 완료되면 멤버 초대 화면으로 전환됩니다.

#### 추천 멤버
- 이전에 함께 달렸던 사용자가 "추천" 배지와 함께 표시됩니다
- 함께한 횟수 정보도 제공됩니다
- 함께한 멤버가 없으면 "아직 함께했던 멤버가 없습니다." 안내가 표시됩니다

#### 닉네임 검색
- 검색창에 닉네임을 입력하여 사용자를 검색할 수 있습니다
- 검색 결과가 없으면 "검색 결과가 없습니다." 안내가 표시됩니다

#### 초대 방법
1. 각 멤버 옆의 **"초대하기"** 버튼을 탭하여 선택합니다
2. 선택된 멤버는 상단에 태그 형태로 표시됩니다
3. 태그의 X 버튼으로 개별 해제할 수 있습니다
4. **"초대 완료"** 버튼으로 초대를 확정합니다
5. **"건너뛰기"**로 초대 없이 벙 생성을 완료할 수도 있습니다

---

## 벙 정보 항목 설명

### 페이스란?
- 1km를 달리는 데 걸리는 시간을 의미합니다
- "분'초" 형식으로 표시됩니다 (예: 5'30" = 1km당 5분 30초)
- 벙의 목표 러닝 속도를 나타냅니다

### 뒷풀이란?
- 러닝 후 함께하는 회식이나 모임을 의미합니다
- "유"를 선택하면 뒷풀이 내용(장소, 메뉴 등)을 추가로 입력할 수 있습니다

### 해시태그란?
- 벙의 특성을 나타내는 키워드입니다
- 예: #초보환영, #한강러닝, #야간러닝 등
- 다른 사용자가 해시태그로 벙을 검색할 수 있습니다
