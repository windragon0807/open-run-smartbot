# 벙 상세 및 관리

## 벙 상세 페이지

벙 카드를 탭하면 해당 벙의 상세 페이지로 이동합니다.

### 표시 정보

- **대표 이미지**: 상단에 벙의 메인 이미지가 크게 표시됩니다
- **남은 시간 배너**: 참여 중인 사용자에게만 표시됩니다
  - 벙 시작 전: 분홍색 배경에 카운트다운 + "시작까지 남은 시간"
  - 벙 진행 중: 초록색 배경에 "Run Started!"
- **벙 이름**: 벙의 제목
- **장소**: 러닝 장소 (위치 아이콘과 함께)
- **시작 날짜/시간**: "M월 d일 (요일) 오전/오후 h:mm" 형식
- **거리 및 페이스**: "X km 페이스" 형식
- **참여 인원**: "현재 인원/최대 인원" + "N자리 남았어요" 뱃지
- **참여 멤버 리스트**: "N명이 함께 뛸 예정이에요" 제목과 함께 멤버 아바타, 닉네임 표시 (벙주는 왕관 아이콘)
- **벙 설명**: 벙에 대한 소개 텍스트
- **뒷풀이 정보**: 뒷풀이가 있는 경우에만 표시
- **Google 지도**: 벙 위치가 지도에 마커로 표시됩니다
- **해시태그**: 벙에 설정된 해시태그 뱃지들

---

## 역할별 사용 가능한 기능

### 참여하지 않은 사용자

- **참여하기 버튼**: 벙에 참여할 수 있습니다
  - 자리가 없으면(정원 초과) 버튼이 비활성화됩니다
  - 확인 다이얼로그 후 참여가 완료됩니다

### 참여한 일반 멤버

- **참여 취소**: 헤더 우측의 "참여 취소" 버튼으로 벙 참여를 취소할 수 있습니다
- **참여 인증**: 벙 장소 근처에서 GPS 기반 참여 인증을 할 수 있습니다
  - 이미 인증을 완료한 경우 버튼이 비활성화됩니다
- **"참여 인증을 왜 해야 하나요?"**: 참여 인증에 대한 안내를 확인할 수 있습니다

### 벙주(방장)

벙을 만든 사람은 벙주로서 다음 기능을 사용할 수 있습니다:

- **벙 수정** (연필 아이콘): 벙 정보를 수정합니다
- **벙주 넘기기** (교체 아이콘): 다른 멤버에게 벙주 권한을 위임합니다
- **벙 삭제** (휴지통 아이콘): 벙을 삭제합니다
- **멤버 관리**: 멤버를 관리(내보내기)할 수 있습니다
- **벙 완료**: 벙을 완료 처리합니다
  - 벙 시작 시간이 지나야 활성화됩니다
  - 참여 인원이 2명 이상이어야 완료할 수 있습니다
  - 시작 전에는 "M/d 오전/오후 h:mm 이후에 버튼이 활성화됩니다" 안내가 표시됩니다

## 참여 인증

### 참여 인증이란?
실제로 벙 장소에 왔는지 GPS를 통해 확인하는 기능입니다.

### 인증 방법
1. 벙 상세 페이지에서 **"참여 인증"** 버튼을 탭합니다
2. 지도에 현재 위치(파란 마커)와 벙 목적지(빨간 마커)가 동시에 표시됩니다
3. **목적지로부터 500m 이내**에 있으면 "참여 인증 완료" 버튼이 활성화됩니다
4. 500m 밖에 있으면 "목적지까지 500m 이내여야 합니다." 안내가 표시되며 버튼이 비활성화됩니다
5. 인증 성공 시 "참여 인증 성공!" 알림이 표시됩니다

### 인증 거리 기준
- **500m 이내**: 벙 장소로부터 반경 500m 이내에 있어야 참여 인증이 가능합니다

## 벙 수정

벙주만 벙 정보를 수정할 수 있습니다.

### 수정 가능한 항목
| 항목 | 설명 |
|------|------|
| 벙 이름 | 벙의 제목 변경 |
| 설명 | 벙 소개 문구 변경 |
| 참가 인원 | 최대 인원 변경 (2~300명, 현재 참여 멤버 수보다 적게 설정 불가) |
| 뒷풀이 | 뒷풀이 유/무 및 설명 변경 |
| 해시태그 | 태그 추가/삭제 |
| 대표 이미지 | 랜덤 버튼으로 이미지 변경 |

### 수정 불가능한 항목
- 장소, 시작 일시, 예상 시간, 거리, 페이스는 수정할 수 없습니다

## 벙주 넘기기 (벙주 위임)

벙주가 다른 멤버에게 벙주 권한을 양도하는 기능입니다.

### 이용 조건
- 벙 시작 **1시간 전까지** 양도 요청이 가능합니다
- 지정한 멤버가 **30분 이내**에 요청을 수락하면 양도가 완료됩니다

### 위임 방법
1. 벙 상세 페이지에서 벙주 넘기기 아이콘을 탭합니다
2. 멤버 목록에서 닉네임 검색으로 원하는 멤버를 찾습니다
3. 해당 멤버 옆의 **"벙주 넘기기"** 버튼을 탭합니다
4. 확인 모달에서 최종 확인합니다

---

## 멤버 관리

벙주만 멤버를 관리할 수 있습니다.

### 기능
- 멤버 목록에서 닉네임 검색으로 멤버를 찾을 수 있습니다
- 벙주가 아닌 멤버를 **"내보내기"** 할 수 있습니다
- 내보내기 시 확인 모달이 표시됩니다
- 벙주 자신은 내보내기 대상에서 제외됩니다

---

## 벙 완료

벙주가 러닝 모임을 완료 처리하는 기능입니다.

### 완료 조건
- 벙 시작 시간이 지나야 완료 버튼이 활성화됩니다
- 참여 인원이 **2명 이상**이어야 완료할 수 있습니다

### 완료 후 좋아요 피드백
벙이 완료되면 좋아요 피드백 화면이 표시됩니다:

1. "벙 완료!"라는 제목과 함께 "함께 달렸던 멤버들에게 👍를 남겨보세요" 안내가 표시됩니다
2. 자기 자신을 제외한 멤버 목록이 표시됩니다
3. 각 멤버 옆의 엄지 아이콘을 탭하여 좋아요를 선택/해제할 수 있습니다 (다중 선택 가능)
4. **"저장"** 버튼으로 좋아요를 전송합니다
5. 좋아요를 받은 멤버의 인기도(좋아요 수)가 올라갑니다
