# 오픈런(OpenRun) 서비스 개요

## 오픈런이란?

오픈런은 러닝 모임("벙")을 만들고 참여하는 소셜 러닝 서비스입니다. 사용자들은 함께 달릴 모임을 개설하거나 다른 사람이 만든 모임에 참여할 수 있으며, 도전과제를 달성하면 NFT 보상을 받아 자신만의 아바타를 꾸밀 수 있습니다.

## 핵심 기능 요약

### 1. 벙(러닝 모임) 시스템
- 러닝 모임을 직접 만들거나 다른 사람의 모임에 참여
- 장소, 시간, 거리, 페이스 등 러닝 정보를 설정
- GPS 기반 참여 인증 (목적지 500m 이내)
- 벙 완료 후 함께 달린 멤버에게 좋아요 피드백

### 2. 탐색 및 검색
- 추천 벙 목록 제공
- 멤버 닉네임, 해시태그, 위치 기반 벙 검색

### 3. 도전과제 시스템
- 일반 도전과제: 특정 조건 달성 시 완료
- 반복 도전과제: 매일/매주 반복 수행하여 스테이지별 달성
- 이벤트 도전과제: 기간 한정 특별 도전과제
- 달성 시 NFT 보상 획득

### 4. 아바타 시스템
- NFT로 획득한 아이템으로 캐릭터 꾸미기
- 상의, 하의, 신발, 얼굴, 피부, 머리카락, 액세서리 등 다양한 카테고리
- 일반(Common), 레어(Rare), 에픽(Epic) 3단계 희귀도

### 5. 프로필
- 받은 좋아요 수, 개설한 벙 수, 획득한 NFT 수 확인
- 최근 획득 NFT 및 완료한 벙 목록 확인

## 서비스 핵심 루프

1. **벙 생성** → 러닝 모임을 만들고 멤버를 모집
2. **벙 참여** → 다른 사람의 모임에 참여
3. **GPS 참여 인증** → 실제 장소에서 500m 이내에서 참여 인증
4. **벙 완료** → 벙주가 벙을 완료 처리
5. **좋아요 피드백** → 함께 달린 멤버에게 좋아요
6. **도전과제 달성** → 활동을 통해 도전과제 진행
7. **NFT 보상 획득** → 도전과제 달성 시 NFT 아이템 획득
8. **아바타 꾸미기** → 획득한 NFT로 캐릭터 커스터마이징

## 로그인 방식

오픈런은 블록체인 스마트 월렛을 통해 로그인합니다.

- 개인 정보를 수집하지 않으며, 지갑 주소만으로 인증합니다
- 로그인 시 금융 거래가 발생하지 않습니다
- 서명은 지갑이 본인 것임을 확인하는 인증용 디지털 서명입니다
- NFT 보상을 받기 위한 전용 월렛이 자동으로 생성됩니다

## 접근 환경

- **모바일 앱**: React Native 앱 내 WebView로 동작
- **웹 브라우저**: 모바일 웹 브라우저에서도 접근 가능
- 데스크톱에서 접속 시 마케팅 페이지가 표시됩니다
