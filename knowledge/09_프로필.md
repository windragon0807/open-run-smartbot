# 프로필

프로필 페이지는 하단 네비게이션의 다섯 번째(마지막) 탭에 해당합니다. 내 정보와 활동 내역을 확인할 수 있습니다.

## 프로필 화면 구성

### 기본 정보
- **프로필 이미지**: 현재 착용 중인 아바타 이미지가 중앙에 표시됩니다
- **닉네임**: 사용자의 닉네임
- **지갑 주소**: 축약된 블록체인 지갑 주소 (탭하면 클립보드에 복사)

### 활동 통계 (3가지)
| 항목 | 아이콘 | 설명 |
|------|--------|------|
| 받은 좋아요 | 엄지 아이콘 | 다른 사용자로부터 받은 좋아요 총 수 |
| 개설한 벙 | 왕관 아이콘 | 내가 만든 벙의 총 수 |
| 획득한 NFT | 깃발 아이콘 | 도전과제로 획득한 NFT 총 수 |

### 최근 획득 NFT 슬라이더
- 3초 간격으로 자동 슬라이드되는 NFT 카드
- 각 슬라이드에는 NFT 이미지, "최근 획득한 NFT" 라벨, 제목, 설명, 획득 날짜가 표시됩니다

### 완료한 벙 리스트
- 스크롤 가능한 완료된 벙 목록
- 각 항목에는 벙 제목, 장소, 날짜가 표시됩니다
- **"피드백 남기기"** 버튼으로 완료한 벙에 대한 피드백을 작성할 수 있습니다

## 프로필 관련 기능

### 아바타 변경
- 프로필 이미지 옆의 **옷 아이콘**을 탭하면 아바타 변경 페이지로 이동합니다

### 설정
- 우측 상단의 **톱니바퀴 아이콘**을 탭하면 설정 모달이 열립니다

---

## 설정 모달

설정 모달에서는 다음 기능을 사용할 수 있습니다:

### 사용자 정보 수정
- 닉네임, 평균 페이스, 주간 러닝 빈도를 수정할 수 있습니다
- 사용자 정보 수정 페이지로 이동합니다

### 알림 설정
- 알림 관련 설정을 변경할 수 있습니다
- 알림 설정 페이지로 이동합니다

### 회원 탈퇴
- 서비스 탈퇴를 진행할 수 있습니다
- 탈퇴 확인 모달이 표시되며, 최종 확인 후 탈퇴가 완료됩니다

---

## 사용자 정보 수정

프로필 설정에서 다음 정보를 수정할 수 있습니다:

| 항목 | 설명 |
|------|------|
| 닉네임 | 서비스에서 사용하는 이름 (중복 확인 필요) |
| 평균 페이스 | 1km당 소요 시간 (분'초" 형식) |
| 주간 러닝 빈도 | 일주일에 달리는 횟수 (0~7회) |

---

알림 수신 여부를 설정할 수 있습니다.
